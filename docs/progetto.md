# BankManager – Gestione Bancaria in C++

## Descrizione
BankManager è un'applicazione console scritta in C++ che simula la gestione di una banca:
- Gestione del capitale della banca.
- Creazione di clienti con patrimonio personale.
- Sistema di prestiti e ricoprimenti con guadagno del 22%.
- Menu interattivo per operazioni in tempo reale.

## Obiettivi del progetto
- Applicare i principi di programmazione orientata agli oggetti (OOP).
- Simulare un sistema economico semplice ma realistico.
- Gestire input utente e condizioni di errore in modo sicuro.
